# = AliveGR Kodi addon git repository =
## Your one addon for everything greek related

![](https://raw.githubusercontent.com/Twilight0/plugin.video.AliveGR/master/resources/media/icon.png)

- Live TV
- Radio
- Video on demand
- News
- Movies
- Kids
- Documentaries
- Sports
- Music on demand
- TV Series & TV Shows
- Short Films

- Windows 10 64 bit
- Ubuntu Linux 64 bit
- Android 5, 7 & 8

**Kodi Versions tested:**

- Leia 18.X (x64)
